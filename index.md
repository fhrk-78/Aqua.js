# Aqua.js ドキュメント
## 何これ？
これはHTML上でAquaを動かすフレームワークです。

Aquaタグで囲った部分がインタープリタによって実行されます
```html:helloworld
<aqua>
    outf "HelloWorld!"
</aqua>
```
## Aquaって?
sakamotor様によって作られた言語です。

元々はコンソール上で動作するインタープリタ言語です
